Quick Installation

                
Set up the plugin in three simple steps.

            


            

                

                    
1

                    

                        
Download the Release Zip

                        
Download the latest pre-packaged release make-it-rain.zip from the GitHub Releases tab.

                    

                

                

                    
2

                    

                        
Extract to Plugins Directory

                        
Unzip the files and place the folder directly into your Obsidian vault's plugins folder:

                        
.obsidian/plugins/make-it-rain/

                    

                

                

                    
3

                    

                        
Enable & Setup

                        
In Obsidian, go to Settings → Community Plugins, enable Make It Rain, add your API Token, and import your bookmarks!
